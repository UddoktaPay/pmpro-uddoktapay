Integrate UddoktaPay payment gateway with Paid Memberships Pro for accepting payments in Bangladesh.

== Description ==

The UddoktaPay Gateway add-on for Paid Memberships Pro allows you to accept payments through various Bangladeshi payment methods including bKash, Nagad, Rocket, and more.

= Features =
* Accept payments through multiple Bangladeshi payment methods
* Seamless integration with Paid Memberships Pro
* Automatic payment verification
* Support for both one-time and recurring payments
* Customizable payment button text
* Easy to configure

= Requirements =
* WordPress 6.2 or higher
* PHP 7.4 or higher
* Paid Memberships Pro plugin
* UddoktaPay merchant account

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/pmpro-uddoktapay`, or install through WordPress plugins screen
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Memberships → Settings → Payment Gateway & SSL
4. Select "UddoktaPay" as your Payment Gateway
5. Enter your UddoktaPay API credentials
6. Save the settings

== Frequently Asked Questions ==

= Do I need a UddoktaPay account? =

Yes, you need a UddoktaPay merchant account to use this plugin. Visit [UddoktaPay](https://uddoktapay.com) to create an account.

= Which payment methods are supported? =

The plugin supports all payment methods available through UddoktaPay, including:
* bKash
* Nagad
* Rocket
* Other Bangladeshi payment methods

= Is SSL required? =

Yes, SSL is required for accepting payments securely through UddoktaPay.
